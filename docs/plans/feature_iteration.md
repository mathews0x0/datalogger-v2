# Feature Iteration Queue

**Date:** 2026-02-08  
**Goal:** Polish and enhance UX/quality through iterative improvements

---

## Round 1: Essential UX Polish

### 1.1 Empty States
- Show helpful messages when lists are empty
- "No sessions yet? Connect your device to start logging!"
- CTAs to guide user action

### 1.2 Loading Skeletons
- Replace "Loading..." with animated placeholder cards
- Feels faster and more professional

### 1.3 Session Notes
- Add notes field to sessions (rider's own thoughts)
- Weather, setup, goals, observations
- Editable from session detail

### 1.4 User Avatars
- Profile photo display with initials fallback
- Show in headers, leaderboards, comments

### 1.5 Improvement Trends Chart
- Line chart showing best lap progression over time
- Per-track trend visualization
